1.异常类型

  • point outliers（点异常）：这种异常样本往往以单个点的形式存在，其数值相较于整体样本而言是明显异常的（例如异常大或异常小的全局异常）。

  • contextual outliers（上下文异常）：contextual outlier的异常体现在某一段时间区间。例如下图的第二幅子图，该异常点相对于整体样本而言并无异常，但如果考虑其相邻时间内的样本，其有明显的异常特性。

  • collective outliers（集体异常）：指一段时间序列数据为异常。且该段数据中单独看每个样本点都不是异常的，样本的异常性体现在该段数据整体而言是异常的。

2.集体异常检测

检测集体（模式）异常的常见做法：

  • 不和谐分析：利用滑动窗口将时间序列分割成多个子序列，并计算子序列之间的距离（例如，欧几里德距离）以找到时间序列数据中的不和谐，如矩阵配置文件、HotSAX。
  • 子序列聚类：也将子序列分割应用于时间序列数据，并采用子序列作为每个时间点的特征，其中滑动窗口的大小是特征的数量。然后，采用无监督机器学习方法，例如聚类（例如，Kmeans，PCA）或逐点异常值检测算法来检测模式异常值。
    其中，基于子序列的方法有三种不同的检测策略：
    基于模型的方法：每个时间序列被转换为模型参数，然后选择合适的模型距离和聚类算法(通常是传统的聚类算法)，并将其应用于提取的模型参数。
    基于特征的方法：通常在这种方法中，先从每个时间序列中计算等长特征向量，然后进行欧氏距离测量。
    基于形状的方法：通常直接处理原始时间序列数据。依据序列间的距离/相似度度量进行聚类

    3.集体异常检测实战

    检测策略：基于特征的子序列聚类，PCA提取重要特征，DBSCAN算法实现聚类。
    检测流程：
  1. 加载原始数据
  2. 为了提高检测效果，对数据进行滤波平滑处理。
  3. 去除数据趋势影响。
  4. 按照实际数据模式，划分子序列，并提取特征。
  5. 特征过滤，PCA降维。
  6. DBSCAN聚类，检测异常序列。
    首先加载必要的工具并加载数据。
    ```python
    import numpy as np
    import os
    import pandas as pd
    from scipy import signal
    import random as rd
    import datetime
    import matplotlib.pyplot as plt
    import seaborn as sns
    import matplotlib.dates as mdate
    import warnings
    warnings.filterwarnings(“ignore”)
    from tsfresh.utilities.dataframe_functions import roll_time_series
    from statsmodels.tsa.seasonal import seasonal_decompose
    from sklearn.preprocessing import StandardScaler
    from sklearn.feature_selection import VarianceThreshold
    from tsfresh import extract_features
    from scipy import signal
    from sklearn.decomposition import PCA
    from sklearn.neighbors import NearestNeighbors
    from sklearn.cluster import DBSCAN
    from sklearn import metrics
    import plotly.graph_objs as go

#加载资源文件数据
data = pd.read_csv(‘./Data.csv’)
data.index = pd.to_datetime(data.Datetime)

使用低通滤波器滤波。
```python
b, a = signal.butter(10,0.026 , 'lowpass') #配置滤波器 8 表示滤波器的阶数
data['Hu1_0'] = signal.filtfilt(b, a, data['Hu_0']) #data为要过滤的信号
#可视化
plt.figure(figsize=(20,5))
plt.title('Hu_0')
plt.plot(data['Hu1_0'])

结果曲线图如下：

利用时间序列分解，去除曲线趋势。在前期数据分析时找出数据存在两个周期，77min和1day。

# 去趋势
decompose_result = seasonal_decompose(data['Hu1_0'], model="additive", period=77)
# 分离77周期
data["detrend_77"] = data['Hu1_0'] - decompose_result.seasonal
#分解1440周期
decompose_result = seasonal_decompose(data["detrend_77"] , model="additive", period=1440)
# 趋势分量
data["Hu0_trend"] = decompose_result.trend
#删除含空值的行，前面分解的序列前端末端含有空值。
data = data.dropna(axis=0)
#去除趋势
data['detrend_Hu0'] = data['Hu1_0'] - decompose_result.trend
# 可视化
plt.figure(figsize=(20,10))
plt.title('Hu_0')
plt.plot(data['detrend_Hu0'])
接下来按照77窗口大小划分子序列，无重复窗口，并提取每个子序列的特征。使用tsfresh工具包，按照API要求，在表中添加id列。 ```python data['id'] = 1 # 滚动窗口 #max_timeshift:最大偏移量，min_timeshift：最小偏移量，rolling_direction：每次移动的大小和方向。column_sort:按什么排序，默认已从小到大排好 df_rolled = roll_time_series(data, column_id="id", column_sort="time",max_timeshift = 76 ,min_timeshift = 76,rolling_direction=-77) # 特征提取 df_features = extract_features(df_rolled.drop("time",axis = 1), column_id="id") ``` 利用方差和相关系数进行特征过滤。 ```python #删除含空值列 df_features_dropna_All = df_features.dropna(axis=1) # 标准化 scaler1 = StandardScaler() df_features_Standar = pd.DataFrame(scaler1.fit_transform(df_features_dropna_All),index=df_features_dropna_All.index, columns=df_features_dropna_All.columns) df_features_Standar # 特征过滤 filter方差 X_fratures_columns = df_features_Standar.columns selector = VarianceThreshold(0.9) X_features = selector.fit_transform(df_features_Standar) df_features_filter = pd.DataFrame(X_features,index=df_features_Standar.index, columns=X_fratures_columns[selector.get_support(indices=True)]) # 特征过滤 相关系数corr # 剔除相关性系数高于threshold的corr_drop # df_filter.corr() corr_features = df_features_filter.corr() threshold = 0.4 upper_features = corr_features.where(np.triu(np.ones(corr_features.shape), k=1).astype(np.bool)) corr_features_drop = [column for column in upper_features.columns if any(upper_features[column].abs() > threshold)] df_features_filter = df_features_filter.drop(corr_features_drop,axis=1) df_features_filter ``` 过滤后的特征有14个，再使用PCA特征降维。 ```python # PCA降维,目标维数为7。 pca = PCA(n_components=7) pca.fit(df_features_filter) #获取特征主成分占比 variance = pca.explained_variance_ratio_ var=np.cumsum(np.round(variance, 3)*100) plt.figure(figsize=(12,6)) plt.ylabel('% Variance Explained') plt.xlabel('# of Features') plt.title('PCA Analysis') plt.ylim(0,100.5) plt.plot(var) ``` 画出特征主成分占比曲线图。
选择占比最大的三个特征。 ```python pca = PCA(n_components=3) pca.fit(df_features_filter) pca_scale = pca.transform(df_features_filter) pca_df = pd.DataFrame(pca_scale, columns=['pc1', 'pc2', 'pc3']) print(pca.explained_variance_ratio_) pca_df ```
查看特征3D图。 ```python Scene = dict(xaxis = dict(title = 'PC1'),yaxis = dict(title = 'PC2'),zaxis = dict(title = 'PC3')) trace = go.Scatter3d(x=pca_df.iloc[:,0], y=pca_df.iloc[:,1], z=pca_df.iloc[:,2], mode='markers',marker=dict(colorscale='Greys', opacity=0.6, size = 10)) layout = go.Layout(margin=dict(l=0,r=0),scene = Scene, height = 1000,width = 1000) data = [trace] fig = go.Figure(data = data, layout = layout) fig.show() ```
DBSCAN需要确定两个参数：epsilon：聚类半径；minPts：聚类点数。其中minPts一般取特征数目的两倍，对应项目中取6。epsilon利用肘形图确定。 画出特征的肘形图，即在y轴上，绘制点之间的平均距离，在x轴上绘制数据集中的所有数据点。从图中，取肘部数值约1.4。 ```python plt.figure(figsize=(10,5)) nn = NearestNeighbors(n_neighbors=6).fit(pca_df) distances, idx = nn.kneighbors(pca_df) distances = np.sort(distances, axis=0) distances = distances[:,1] plt.plot(distances) plt.show() ```
确定完两个参数后，开始聚类。输出结果为聚类簇数、噪声点数和剪影得分（评价聚类效果，越接近1越好）。 ```python db = DBSCAN(eps=1.4, min_samples=6).fit(pca_df) labels = db.labels_# Number of clusters in labels, ignoring noise if present. n_clusters_ = len(set(labels)) - (1 if -1 in labels else 0) n_noise_ = list(labels).count(-1) print('Estimated number of clusters: %d' % n_clusters_) print('Estimated number of noise points: %d' % n_noise_) print("Silhouette Coefficient: %0.3f" % metrics.silhouette_score(pca_df, labels)) ``` 输出结果： Estimated number of clusters: 1 Estimated number of noise points: 10 Silhouette Coefficient: 0.528 将筛选出来的噪声点即异常点反映在3D图上。 ```python Scene = dict(xaxis = dict(title = 'PC1'),yaxis = dict(title = 'PC2'),zaxis = dict(title = 'PC3')) labels = db.labels_ trace = go.Scatter3d(x=pca_df.iloc[:,0], y=pca_df.iloc[:,1], z=pca_df.iloc[:,2], mode='markers',marker=dict(color = labels, colorscale='Viridis', size = 10, line = dict(color = 'gray',width = 5))) layout = go.Layout(scene = Scene, height = 1000,width = 1000) data = [trace] fig = go.Figure(data = data, layout = layout) fig.update_layout(title='DBSCAN clusters Derived from PCA', font=dict(size=12,)) fig.show() ```
